# Machine Learning Learning Notes

This repository records my hands-on practice while studying *Introduction to Machine Learning with Python*.

My goal is to build a solid foundation in classical machine learning and prepare for machine learning engineer interviews.

## Topics

- Supervised Learning
- Unsupervised Learning
- Preprocessing
- Feature Engineering
- Model Evaluation
- Pipelines and Grid Search

## Repository Structure

- `notebooks/`: Jupyter notebooks for book examples and experiments
- `mini_projects/`: Small machine learning projects based on what I learn
- `requirements.txt`: Python packages required for this repository

## Learning Goal

For each topic, I will try to understand:

1. What problem the model solves
2. What the input and output are
3. How to train the model
4. How to evaluate the model
5. How to explain the result in interview-style language
